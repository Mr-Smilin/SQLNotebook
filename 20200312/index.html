正規化TABLE中取非主鍵彙總


表A
主鍵:A1 A2
B1 B2 B3
 

表B
主鍵:C1 C2 C3
D1 D2 D3
 
 
SQL 1
 
SELECT 表A.*
FROM 表A
 
 
SQL 2
 
SELECT D1 D2 D3
FROM 表B
WHERE C1=SQL1.B1 AND C2=SQL1.B2

相同B1,B2,D1下的D2與D3各自彙總後印出
 

A1,A2,C1,C2,C3,D1,D2,D3
